# Basic Workflow
## `git init`
- It initializes a new Git repository.

## `git status`
- It shows working directory status.


## `git add <yourfile>`
- It add and Stages changes for commit.

## `git commit -m " your ref. message "`
- It records staged changes.


---

# Viewing Changes

## `git log`
- It shows commit history.
